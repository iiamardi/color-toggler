# Color Toggler App

A simple React application that allows users to dynamically change the background color of the screen. This project was built to practice state management and event handling in React.

## Live Demo

[View Live Demo](https://color-toggler-kg7g.vercel.app/)

## Features

- Change background color with a click
- Interactive UI updates in real time
- Simple and clean design
- Built with React hooks (`useState`)

## Tech Stack

- React
- JavaScript (ES6+)
- CSS

## What I Learned

- Using `useState` for dynamic UI changes
- Handling button click events in React
- Conditional rendering and UI updates
- Basic component structure in React
